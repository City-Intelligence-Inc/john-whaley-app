# ATS Site Search

[[Jordan Mazer]]'s job search hack using search engines to find unlisted jobs.

## How It Works
Go to Google and search:
```
site:boards.greenhouse.io ("founding engineer" AND "remote")
```

Or for designers:
```
site:boards.greenhouse.io ("UX designer" OR "UI designer") AND "remote"
```

## Why It Works
- Less competition — most candidates don't know this
- Finds jobs before they're widely shared
- Works with boolean operators for precision
